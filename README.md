# Customer Segmentation Dashboard

## 📌 Project Overview

This project focuses on customer segmentation analysis using Power BI. The dashboard helps identify customer purchasing behavior, spending patterns, and regional customer distribution through interactive visualizations.

---

## 🚀 Features

* Customer Sales Analysis
* Top 10 Customers Visualization
* Region-wise Customer Distribution
* Interactive Filters (Slicers)
* Customer Spending Insights
* KPI Cards for Total Sales

---

## 🛠 Tools Used

* Power BI
* Excel / CSV Dataset

---

## 📊 Dashboard Insights

* Identified high-value customers
* Compared customer sales across different regions
* Visualized customer purchasing patterns
* Analyzed customer distribution effectively
